# 电脑硬件推荐系统 python1191

## 项目概述

本项目是一款基于Python大数据的电脑硬件推荐系统。通过收集和处理大量的电脑硬件数据，为用户提供个性化的电脑硬件推荐。

## 技术栈

- **后端框架**：Spring Boot
- **数据库**：MySQL
- **前端框架**：VUE

## 功能模块

### 管理员模块

管理员是系统的核心用户，负责系统后台的管理。

- 用户管理
- 品牌管理
- 笔记本管理
- 电脑主机管理
- 电脑外设管理
- 硬件组装管理
- 电脑信息管理
- 系统管理

### 用户模块

用户可在系统中管理个人中心和收藏等功能。

- 个人中心管理
- 我的收藏管理

## 系统角色

- 管理员：负责系统后台的整体管理和维护。
- 用户：使用系统的推荐功能，管理个人账户信息。

## 运行环境

- 后端：Java 1.8+，Spring Boot 2.x
- 数据库：MySQL 5.7+
- 前端：Node.js，VUE 2.x

## 部署步骤

1. 准备Java和MySQL的运行环境。
2. 导入MySQL数据库脚本。
3. 部署后端Spring Boot应用程序。
4. 部署前端VUE应用程序。
